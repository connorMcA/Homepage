Description

Q'dUp is an appplication that makes it easy for everyone to play the music they want to hear at the party. One person simply hosts a queue on their phone and gets a code which they can give to their friends who can then join and add to the queue. It's as simple as that.

The app is being developed in Java utilizing Spotify's API to search, access, and play songs. The app currently allows users to both host and join a queue, being able to add songs in either case. The host user also has the ability to rearrange the queue order, as well as remove songs. In the works is migrating the backend server to be hosted on AWS and updating some frontend UI/UX. We hope to release version 1.0.0 to the app store soon.
